 \section{Análisis de Rendimiento}
 		Una vez terminado el IP Core para comprobar su correcto funcionamiento y ver si éste realmente 
 		tiene una mejora en los tiempos de sincronización, se realizaron mediciones para distinto número
 		de iteraciones, tanto para el Procesador de Petro como para Semáforos. La elección de este segundo
 		método de sincronización se basa en que son el mecanismo más ligero para realziar éstas tareas.
 		A partir de estas mediciones se realizó la gráfica de la Figura \ref{fig:resultados13} 
 		
		\begin{figure}[h]
			\centering
			\includegraphics[width=3in]{./img/resultados13}
			\caption{Tiempos de sincronización procesador de Redes de Petri vs. Semáforos}
			\label{fig:resultados13}
		\end{figure}		
		
		Se puede observar que, para todos los casos, el procesador de Petri es aproximadamente cuatro veces
		más rapido que los Semáforos.
